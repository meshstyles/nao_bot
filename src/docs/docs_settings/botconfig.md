## botconfig.json

### config
#### token
discord token
#### vote
vote command prefix
#### status
bot status command prefix
#### join
role join command prefix
#### leave
leave join command prefix
#### vc_create
voice channel command prefix
#### vc_remove
voice channel command prefix

### responses
#### res_dm
response for DMs
#### pol_failed
response for a failed poll initialization
#### pol_toManyArgs
response for to many poll answers

### splitters
#### votesplit
splits question poll
#### votesplit2
splits ansewers poll

a sample is provided in ./src/settings/botcomfig_sample.json
